13. 전 기능 엑셀 익스포트

다른 기능 화면 상단에도 동일한 버튼이 있으며, 여기서는 통합 추출 동작을 안내합니다. 통합 파일에는 01~15 접두 시트명으로 기능별 데이터가 구분되어 들어갑니다.

「현재 화면 엑셀 저장」은 이 페이지 전용 안내 시트를 보냅니다. 반드시 「1~15 전 기능 통합 엑셀 추출」을 사용해 전체 데이터를 받으세요.
