머니 캘린더 가이드

1) 작성 방법

먼저 1~4번에서 수입과 예산 구조를 설계합니다. 그다음 5~8번에서 하루 지출을 기록하고, 9~12번에서 주간·월간·시계열·피드백으로 흐름을 정리합니다.

2) 관리 방식

데이터는 이 브라우저의 localStorage에 저장됩니다. 필요할 때 엑셀 다운로드/업로드로 데이터를 직접 소유하고, 다시 화면에 불러와 이어서 사용할 수 있습니다.

3) 엑셀 활용

각 화면 상단의 「표준 양식 다운로드」로 템플릿을 받고, 「파일에서 불러오기」로 동일한 양식을 다시 적용하세요. 13번 화면에서는 1~15 데이터를 한 파일로 통합 추출할 수 있습니다.

4) Data & Security

보안 원칙. 귀하의 재정 데이터는 서버로 전송되지 않습니다. 오직 사용자님의 브라우저(내 기기)에만 안전하게 자동 저장됩니다.

데이터 연속성. 로그인 없이도 새로고침이나 브라우저 재시작 시 기록이 유지됩니다. 다른 기기에서 이용하시려면 엑셀 파일로 백업 후 업로드하세요.
